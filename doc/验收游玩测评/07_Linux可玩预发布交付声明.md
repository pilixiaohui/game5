## 2. 可以声明的事实

在上述候选和环境中，独立agent从正式项目入口完成了以下玩家路径：

1. 从干净用户目录开始新游戏，并在保存后完全结束进程、重启和继续。
2. 建设基础房间，自然形成工蜂、战斗单位和固定单位。
3. 选择不同职责节点，观察敌军、结构、战损和速度档产生的阶段变化。
4. 在正常反应时间内打开撤离确认，取消后继续，再次确认撤离。
5. 观察撤离后的敌军恢复和未占领状态，重新进入后完成真实占领。
6. 保存并跨进程恢复房间、资源、储能和节点进展。

技术自动验证与架构审查为上述候选提供逻辑和回归背景，但没有替代这次生产入口单场操作。

- Windows 10/11、Windows导出包、NTFS、中文Windows账户路径或睡眠恢复通过；
- `NOVICE / GENRE / RETURNING / ACCESS`独立真人分层结果；
- PXI、样本量、统计功效、Wilson区间或目标人群体验通过；
- 45/60分钟内容容量、首小时规模、返回动力或长期留存成立；
- 最低硬件、p95/p99帧时间、60分钟内存包络或百万级性能达标；
- 完整分辨率、DPI、应用缩放、窗口/全屏、多显示器或键盘焦点矩阵通过；
- 真实音频设备、物理GPU/显示器/鼠标键盘或无障碍真人验证通过。

Linux单场结果不能外推为Windows发布门，也不能用一个agent重复操作替代不同玩家样本。
