# 异种起源：无尽洪流 - Godot 垂直切片

## 运行

```bash
godot --path .
godot --headless --path . --quit-after 3
```

主场景：`res://scenes/app/Main.tscn`。项目以 540x960 竖屏入口运行。

## 核心规则验证

```bash
godot --headless --path . -s res://tests/core_rules.gd
```

该脚本验证资源不为负、器官升级成本、储备孵化与投放消耗、吞噬度推进、撤离保留兵力、插件改变战斗、离线 12 小时封顶、重置预览与实际收益一致、存档读写。

## 首局观察路径

1. 等待有机原浆自动增长，升级菌毯绒毛和深钻根须。
2. 在战场面板左侧点击蓄兵，观察波次槽位、缺口叉号、敌压阈值和 4% 重置预览目标线。
3. 储备接近阈值后点击中线强攻，观察补位进入、前线推进、敌压削减和螺旋奖励变化。
4. 升级菌毯/深钻根须或装配酸化甲壳后回到战场，观察波次阈值、有效敌压、损耗和构筑图标变化。
5. 在战场右侧点击撤离保全兵力，观察回流箭头和回收点阵。
6. 达到 4% 目标后查看战场目标线和吞噬世界预览，确认突变原收益已经出现。

## 当前边界

这是技术垂直切片和规则样机的游戏化返工版本，不宣称第一版游戏通过；它只把首局核心循环提升为可观察的战线、单位、敌压、补位和损耗反馈。项目仍不包含 Android 正式包、完整实时虫群战场、星际/深海/泰坦/PvP/服务器经济，也没有完整商业 MVP 内容量。数据结构已保留扩展到更多区域、单位、敌人、元素和插件的入口。
